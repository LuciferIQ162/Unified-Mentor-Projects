ASL Alphabet Predictor

🎯 Project Goal

To build a deep learning model for classifying images of the American Sign Language alphabet, enabling communication through technology.

🛠️ Technologies Used

Core: Python, TensorFlow/Keras
Data Handling: ImageDataGenerator

🔬 Methodology

Constructed a Convolutional Neural Network (CNN) with dropout for regularization. Utilized data augmentation to create a more diverse training set from existing images.

Interactive Model Evaluation

The core of this project was training a robust CNN. The chart below visualizes the model's learning progress over 5 epochs, showing how accuracy improved and loss decreased for both training and validation datasets.

Mobile Phone Price Range Predictor

🎯 Project Goal

To develop a classification model that accurately predicts the price range of mobile phones based on their hardware specifications.

🛠️ Technologies Used

Core: Python, Pandas, Scikit-learn
Model: RandomForestClassifier
Deployment: Joblib

🔬 Methodology

Trained a Random Forest Classifier with 100 estimators and evaluated its performance using accuracy, classification report, and a confusion matrix.

Challenges & Key Learnings

This internship significantly strengthened my skills in model evaluation and practical problem-solving. Here are the key takeaways.

Strategic Scope Adjustment

Learned to pivot project scope effectively. The initial challenge of designing a full UI led to a strategic focus on the depth of model analysis over a front-end interface, delivering more impactful results.

Deep Model Evaluation

Gained a critical understanding that true model performance is understood through metrics like confusion matrices and training history, not just a single accuracy score.

Practical Problem-Solving

Developed skills in fine-tuning model hyperparameters to improve accuracy and reduce loss, alongside techniques for managing and preprocessing large image datasets efficiently.
